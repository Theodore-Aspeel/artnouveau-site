# Notice utilisateur - éditeur local

Cette notice sert à faire un premier test simple de l'éditeur local des articles.

## 1. Lancer l'éditeur

1. Ouvrir le dossier du projet.
2. Double-cliquer sur `lancer-editeur.bat`.
3. Garder la fenêtre noire ouverte.
4. Le navigateur doit s'ouvrir tout seul.

Si le navigateur ne s'ouvre pas, copier cette adresse dans le navigateur :

```text
http://127.0.0.1:8765/
```

## 2. Choisir un article

La liste des articles se trouve à gauche.

Cliquer sur le titre d'un article pour l'ouvrir.

## 3. Modifier un article

Modifier uniquement les champs utiles au test.

Les grands champs de texte peuvent être agrandis avec la souris depuis leur coin inférieur droit.

Pour changer une image, choisir une image dans la liste proposée. L'éditeur ne permet pas d'ajouter une nouvelle image depuis l'ordinateur.

## 4. Valider

Cliquer sur `Valider`.

Si un message d'erreur apparaît, corriger le champ indiqué puis cliquer à nouveau sur `Valider`.

La validation ne sauvegarde pas encore les changements.

## 5. Enregistrer

Cliquer sur `Enregistrer`.

Un message confirme que l'enregistrement a réussi.

## 6. Prévisualiser

Après avoir choisi un article, utiliser les boutons :

- `Preview FR` pour voir la version française.
- `Preview EN` pour voir l'aperçu anglais si disponible.

La prévisualisation s'ouvre dans un nouvel onglet.

## 7. Arrêter l'éditeur

Fermer la fenêtre noire ouverte par `lancer-editeur.bat`.

Si elle ne se ferme pas, cliquer dedans puis appuyer sur `Ctrl+C`.

## Limites normales pour ce test

L'éditeur local ne publie pas le site.

Il ne sert pas à ajouter de nouveaux articles, ajouter de nouveaux fichiers image, traduire automatiquement, ou réorganiser les images.
